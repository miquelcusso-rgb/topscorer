# PWA & Play Store Screenshots

Required screenshots for Google Play Store submission:

## Mobile (portrait, 390x844px or similar 9:19.5 ratio)
- screenshot-mobile.png — main stats table on dark mode
- screenshot-mobile-light.png — light mode variant

## Desktop (landscape, 1280x720px)
- screenshot-desktop.png — full dashboard view

## Play Store (required formats)
- feature-graphic.png — 1024x500px banner image
- hi-res-icon.png — 512x512px icon (must have no rounded corners — Google adds them)

## How to capture
Use Chrome DevTools -> Device toolbar -> iPhone 14 Pro (390x844)
Navigate to top-scorers.com, take screenshot with Cmd+Shift+4 on Mac.
